# Automação de Análise de Dados com Python

Projeto de portfólio que automatiza leitura, validação, tratamento e geração
de relatórios a partir de arquivos CSV usando Python.

## Funcionalidades
- Entrada por arquivo ou pasta
- Validação de colunas obrigatórias
- Tratamento e limpeza de dados
- Geração de CSV tratado e Excel com resumo
- Métricas finais no console

## Execução
```bash
pip install -r requirements.txt
python main.py --input data/entrada.csv
